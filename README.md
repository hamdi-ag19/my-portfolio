# Personal Website Portfolio

A modern, responsive portfolio website for Hamdi Al-Luqmani, a Full-Stack Web Developer.

## Project Structure

```
/
├── index.html              # Main homepage
├── contact.html            # Contact form page
├── assets/
│   ├── css/
│   │   ├── Home.css        # Styles for index.html
│   │   └── contact-form.css # Styles for contact.html
│   └── js/
│       ├── Home.js         # Scripts for index.html
│       ├── Translation.js  # Translation functionality
│       └── contact-form.js # Scripts for contact.html
├── favicon.ico             # Site favicon
└── CV Hamdi.pdf            # Resume file
```

## Features

- Bilingual support (English/Arabic) with RTL layout
- Dark/Light mode toggle
- Responsive design
- Contact form with validation
- Smooth animations and transitions
- SEO optimized with structured data

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome icons
- Google Fonts
- Formspree for form handling

## Getting Started

1. Clone or download the repository
2. Open `index.html` in a web browser
3. For development, use a local server to avoid CORS issues

## Customization

- Update URLs in structured data scripts
- Replace placeholder content with your information
- Modify colors and styles in CSS files
- Add your own images and assets

## Deployment

Upload all files to your web hosting service. Ensure the `assets/` folder is included.
